  • TransmitOptions

    The TransmitOptions class controls options in transmitting APDUs.

    The options are initialized with the following values:

    • GET RESPONSE for T=0: Enabled
    • GET RESPONSE for T=1: Enabled

    • Strip Le for T=1: Disabled
